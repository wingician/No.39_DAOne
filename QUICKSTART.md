# 快速开始指南

欢迎来到翼术师世界！这个指南将帮助你快速了解和开始使用这个项目。

## 🎯 项目概述

**翼术师世界**是一个开放式协作的魔法世界创作项目，特点：

- 🌐 **开放协作**：任何人都可以参与世界设定创作
- 🤖 **AI 检查**：自动检查新内容与现有设定的冲突
- 🌳 **分支管理**：支持创建平行宇宙世界线
- 🎨 **魔法主题**：精美的魔法系网站设计

## 🚀 快速开始（5分钟）

### 对于访问者

1. **访问网站**：https://wingician.github.io/doc/
2. **浏览世界观**：了解九神州、魔法体系、历史纪元
3. **阅读故事**：探索已有的角色和故事

### 对于贡献者

1. **Fork 仓库**
   ```bash
   # 在 GitHub 上点击 Fork 按钮
   git clone https://github.com/你的用户名/doc.git
   cd doc
   ```

2. **创建分支**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **添加内容**
   - 在 `故事背景/` 下添加世界设定
   - 在 `人物卡片/` 下添加角色资料
   - 在 `小故事/` 下添加故事创作
   - 使用 Markdown 格式

4. **提交 PR**
   ```bash
   git add .
   git commit -m "Add: 你的内容描述"
   git push origin feature/your-feature-name
   ```
   然后在 GitHub 上创建 Pull Request

5. **等待审核**
   - AI 会自动检查冲突
   - 维护者审核通过后合并

## 📝 内容创作指南

### 世界设定

创建文件：`故事背景/分类/你的设定名.md`

模板：
```markdown
# 设定名称

## 基本信息
- **位置**：所在大陆/区域
- **类型**：城市/地区/事件等
- **时代**：所属纪元

## 描述

详细描述...

## 相关设定

- [[关联设定1]]
- [[关联设定2]]
```

### 人物卡片

创建文件：`人物卡片/角色名.md`

模板：
```markdown
# 角色名

## 基本信息
- **全名**：
- **年龄**：
- **性别**：
- **种族**：
- **职业**：翼术师/法术师等
- **等级**：学徒/耶师/亲师/翼术师

## 外貌特征

## 性格

## 能力

## 背景故事

## 重要关系
```

### 故事创作

创建文件：`小故事/故事标题.md`

确保：
- 符合世界观设定
- 时间线合理
- 人物行为符合性格
- 魔法使用符合理论

## 🤖 AI 检查说明

提交 PR 后，AI 会检查：

1. ✅ **时间线**：事件时间是否合理
2. ✅ **地理**：地点是否存在
3. ✅ **魔法**：魔法理论是否一致
4. ✅ **人物**：角色关系是否合理
5. ✅ **文化**：习俗是否符合设定

如果发现潜在冲突，AI 会在 PR 中评论提示。

## 💡 创作建议

### 好的实践 ✅

- 深入理解东方哲学元素
- 保持"外西内东"的风格
- 详细的背景设定
- 人物有深度和成长

### 避免的问题 ❌

- 引入现代科技（除非在特定分支）
- 过于强大的人物设定
- 与时间线冲突
- 忽略魔法理论限制
- 纯粹的西方奇幻设定

## 🔧 常见问题

### Q: 我不懂编程，可以参与吗？
A: 当然！你只需要会写 Markdown 文档。可以直接在 GitHub 网页上创建和编辑文件。

### Q: 我的内容被AI标记冲突怎么办？
A: 阅读 AI 的建议，修改内容后重新提交。如果认为 AI 误判，可以在 PR 中说明。

### Q: 可以写与主线不同的设定吗？
A: 可以！创建一个新的平行宇宙分支，在那里可以有不同的设定。

### Q: 多久能看到我的内容上线？
A: 合并到 main 分支后，GitHub Pages 会在几分钟内自动部署。

### Q: 我想修改别人的内容怎么办？
A: 提交 PR 说明修改原因。如果是错误修正，通常会被接受；如果是大改，需要详细说明。
