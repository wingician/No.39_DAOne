魔法体系

混沌能量论

魔法是一种来自混沌的能量，术师是能量的通道。通过汲取混沌能量，可以改变这个有序世界。

虚拟世界论

整个世界是虚拟的，魔法是世界代码的命令合集，元神赋予了术师使用这些命令的权限。

本体归一论

世界归于一个本体，所有现象都是本体的具象。魔法是通过理解本质来改变形式的能力。

精力映射论

魔法是在本源和相之间插入映射，让相表达出变化。需要消耗术师的精力去理解和创造。

魔法要素

  • 魔杖系统：龙须魔杖、普通魔杖
  • 结界魔法：空间保护与限制
  • 地域魔法：山魔法与海魔法
  • 魔法强度：不同等级的能量层次
  • 魔法限制：不可测、死信息、维度错乱

历史纪元

远古纪元

距今三百万年前

魔法能量飘散期，还没有成型的魔法能量，自然也还没有翼术师。

上古之战

距今三百万年到十万年前

史前核战造成第一次毁灭。稻一大陆侥幸延续，成熟的魔法诞生，第一位翼术师出现。

中古纪元（魔法纪元）

距今三万年到四千多年前

王国由翼术师担任太师，各王国繁荣安定。修筑西方长城抵御魔地威胁。

英雄纪元

距今四千多年到三四百年前

魔法消退，药物兴起。英雄学院与魔法学院争雄，魔法开始走向没落。

完美万物纪元

距今三四百年前

工业革命开始。长城倒塌，魔物第二次入侵。王国限制魔法，两百年后世间已不存在魔法。

参与创作

这是一个开放式协作的魔法世界。我们欢迎所有对这个世界感兴趣的创作者加入！

你可以创作：世界设定、地理风貌、历史事件、魔法体系、人物故事、小说创作...

1. Fork 仓库

在GitHub上Fork我们的仓库，开始你的创作之旅。

Fork Now

2. 创作内容

按照规范添加你的世界设定或故事。支持Markdown格式。

查看规范

3. 自动检查

我们的AI助手会检查你的内容是否与现有设定冲突。

4. 提交PR

创建Pull Request，经过审核后即可合并到主世界线。

平行宇宙分支

你也可以创建新的分支，探索不同的世界线可能性：

main - 正史主线
