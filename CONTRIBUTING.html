贡献指南

感谢你对翼术师世界的兴趣！这是一个开放式协作的魔法世界创作项目，我们欢迎所有形式的贡献。

你可以贡献什么？

世界设定

  • 地理环境：大陆、岛屿、城市、特殊地貌
  • 社会人文：节日、习俗、文化传统
  • 历史事件：重要战役、文明变迁
  • 魔法体系：新的魔法理论、魔法现象
  • 生物种族：动物、植物、魔物、智慧种族

人物角色

  • 角色卡片：背景、性格、能力、关系
  • 术师资料：等级、专精、成就

故事创作

  • 短篇故事：基于世界设定的独立故事
  • 长篇小说：系列创作
  • 人物传记：角色的完整故事

如何开始

1. Fork 仓库

点击右上角的 Fork 按钮，将仓库复制到你的账户下。

git clone https://github.com/你的用户名/doc.git
cd doc

2. 创建分支

git checkout -b feature/你的功能名称

3. 添加内容

按照文件结构组织你的内容，使用 Markdown 格式编写。

4. 提交更改

git add .
git commit -m "描述你的更改"
git push origin feature/你的功能名称

5. 创建 Pull Request

在 GitHub 上创建 PR，等待自动检查和审核。

自动冲突检查

AI 助手
我们的 AI 助手会自动检查你的提交，确保内容与现有设定不冲突。

检查项包括：

  • 时间线冲突：事件时间、角色出现时间是否合理
  • 地理冲突：地点是否在合理位置
  • 魔法体系冲突：魔法能力是否符合理论
  • 人物冲突：角色关系、能力等级是否合理
  • 文化冲突：习俗是否符合世界设定

内容规范

禁止的内容

  • ❌ 现代科技（电脑、手机、互联网等）
  • ❌ 核武器、现代武器
  • ❌ 与已有魔法理论严重冲突的设定
  • ❌ 打破时间线连贯性的内容

鼓励的内容

  • ✅ 基于东方哲学的魔法理论
  • ✅ 融合易经、五行、八卦的设定
  • ✅ 富有想象力的地理环境
  • ✅ 深入的人物刻画
  • ✅ 引人入胜的故事情节

分支管理

主分支 (main)

主世界线的正史设定，要求最严格。

平行宇宙分支

你可以创建新的分支探索不同的世界线：

  • branch-steampunk - 蒸汽朋克异世界
  • branch-darkage - 黑暗中世纪设定
  • branch-cybermagic - 赛博魔法时代
