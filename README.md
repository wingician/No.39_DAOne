## 🌟 项目简介

**翼术师世界**是一个开放式协作的魔法世界创作项目。这是一个阴阳镜像的世界——外表是西方的奇幻世界，内核却是东方的思想精髓。

在盘古开天辟地时，诞生了现在这个世界的同时，也分裂出另一个宇宙世界——**稻一大陆**。经历**大地动**后，稻一大陆分裂成**九神州**。地球世界的文明破坏会化身为魔王能量在稻一大陆肆虐，而**翼术师**们则要与黑暗魔王斗争，维护两个世界的平衡。

> *翼术师的故事是关于文化和文字的隐喻故事。*

## ✨ 核心特色

### 🎨 独特的世界观
- **阴阳镜像世界**：地球与稻一大陆的双向联系
- **东方哲学内核**：融合易经、五行、八卦、道家思想
- **西方奇幻外壳**：魔法学院、冒险故事、奇幻生物
- **文化隐喻主题**：关于文明、文字、文化传承的深层思考

### 📚 丰富的设定体系

#### 历史纪元
- **远古纪元**（距今三百万年前）：魔法能量飘散期
- **上古之战**（三百万年到十万年前）：第一次毁灭，魔法诞生
- **魔法纪元**（三万年到四千多年前）：翼术师担任太师，王国繁荣
- **英雄纪元**（四千多年到三四百年前）：魔法衰落，英雄学院兴起
- **完美万物纪元**（三四百年前）：工业革命，魔法被禁
- **魔法重现**（四十多年前）：神秘事件...

#### 魔法体系
- **混沌能量论**：魔法来自混沌的能量
- **虚拟世界论**：世界是虚拟的，魔法是代码命令
- **本体归一论**：万物归一，改变形式
- **精力映射论**：在本源和相之间插入映射

#### 地理环境
- **九神州大陆**：稻一大陆分裂后的九大洲
- **凄海**、**狄拉克之海**：神秘的海域
- **魔地**：西方的危险区域
- **南烛大陆**、**错渊大陆**等特色地区

## 🚀 如何参与

### 你可以贡献什么？

- **📍 世界设定**：地理、历史、魔法体系、生物种族
- **👤 人物角色**：角色卡片、术师资料、人物传记
- **📝 故事创作**：短篇故事、长篇小说、故事片段
- **🎨 艺术创作**：插图、地图、角色设计（未来计划）

### 参与流程

1. **Fork 仓库** → 复制项目到你的账户
2. **创建分支** → `git checkout -b feature/your-feature`
3. **添加内容** → 按照规范编写 Markdown 文档
4. **提交 PR** → 等待自动检查和审核

详细指南请查看 [CONTRIBUTING.md](CONTRIBUTING.md)

### 🤖 AI 自动检查

我们的 AI 助手会自动检查你的提交，确保内容与现有设定不冲突：

- ✅ 时间线冲突检查
- ✅ 地理设定验证
- ✅ 魔法体系一致性
- ✅ 人物关系合理性
- ✅ 文化传统冲突

## 🌳 分支管理

### Main 分支
主世界线的正史设定，要求最严格。

### 平行宇宙分支
探索不同的世界线可能性：

- `branch-steampunk` - 蒸汽朋克异世界
- `branch-darkage` - 黑暗中世纪设定
- `branch-cybermagic` - 赛博魔法时代
- 欢迎创建你的平行宇宙！
